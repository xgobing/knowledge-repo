## JVM 参数清单

| 参数名称                | 参数说明         | 默认值                       | 备注                       |
| ----------------------- | ---------------- | ---------------------------- | -------------------------- |
| -Xms                    | 初始堆大小       |                              |                            |
| -Xmx                    | 最大堆大小       |                              |                            |
| -Xss                    | 线程栈容量大小   |                              |                            |
| -XX:PermSize            | 设置永久代初始值 |                              | HotSpot JDK1.7开始不再支持 |
| -XX:MaxPermSize         | 设置永久代最大值 |                              | HotSpot JDK1.7开始不再支持 |
| -XX:MaxDirectMemorySize | 设置直接内存大小 | 不设置默认与Java堆最大值一样 |                            |
